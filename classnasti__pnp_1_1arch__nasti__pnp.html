r_adc_detect std_logic_vector ( 7 downto 0 )
Signal

Detector of the ADC clock.

If this register won't equal to 0xFF, then we suppose RF front-end not connected and FW should print message to enable 'i_int_clkrf' jumper to make possible generation of the 1 msec interrupts.
