Detailed Description

Hard-reset initialization module.

L2-cached system implementing Uncore module must be switched from resetting state that is done by this module. param[in] HTIF interface clock. param[in] Reset signal with the active LOW level. param[in] i_host HostIO input signals. param[out] o_host HostIO output signals.
