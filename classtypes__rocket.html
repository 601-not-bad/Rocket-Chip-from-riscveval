Member Data Documentation

HTIF_WIDTH integer := 16
Constant

Multiplexing HTIF bus data width.

Not used in a case of disabled L2 cache. If L2 cached is enabled this value defines bitwise of the bus between Uncore module and external transievers. Standard message size for the HTID request is 128 bits, so this value defines number of beats required to transmit/recieve such message.

MEM_DATA_BITS integer := CFG_NASTI_DATA_BITS
Constant

Memory data bitwise.

We intentionally select this value equals to AXI4 bus data width. Default SCALA configurator value uses 64/128 SerDes module to access from Tile to AXI4 bus.

MEM_TAG_BITS integer := 6
Constant

Bits allocated for the memory tag value.

This value is defined Config.scala and depends of others configuration paramters, like number of master, clients, channels and so on. It is not used in VHDL implemenation.

NoC global reset former.

Bus clock. param[in] inPllLock PLL status. param[out] outReset Output reset signal with active 'High' (1 = reset).

Starter
Component

Rocket Cores hard-reset initialization module.

Everytime after hard reset Rocket core is in resetting state. Module Uncore::HTIF implements writting into MRESET CSR-register (0x784) and not allowed to CPU start execution. This reseting cycle is continuing upto external write 0-value into this MRESET register. param[in] clk Clock sinal for the HTIFIO bus. param[in] nrst Module reset signal with the active Low level. param[in] i Input interconnect signals. param[out] o Output interconnect signals.
